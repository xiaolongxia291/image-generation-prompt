# 任务
你将看到一张图片。请对图片进行深度视觉拆解，并直接生成一段可用于生图工具的**中文提示词**。

# 目标
尽可能精准还原原图，重点复刻：

- **图片中的具体文字内容**
- **文字所属语言**
- **字体的造型、材质与风格**
- **文字的排版位置与画面占比**
- **与文字相关的背景、装饰、纹理、光影和整体质感**

# 执行顺序
请严格按以下顺序组织提示词内容，并整合为**一段连续中文提示词**：

## 1. 先写清文字内容
这是最高优先级，必须放在最前面。  
明确指出画面中出现的**具体汉字、英文、数字或其他文本**，并描述：

- 写了什么字或词
- 位于哪里
- 占画面多大比例
- 是横排、竖排、倾斜、穿插还是分层排列

## 2. 再细写字体本身
详细描述字体的：

- 风格类型
- 笔画特征
- 结构特征
- 材质表现
- 工艺效果

例如飞白、断裂、拉伸、金属倒角、立体挤压、霓虹发光、磨损裂纹、浮雕镂空等。

## 3. 补充画面细节
围绕文字继续描述：

- 装饰元素
- 背景环境
- 前景遮挡
- 纹理颗粒
- 主体与背景的互动关系

如果原图局部模糊，可根据画面逻辑进行**高精度补全**，但不能偏离原图风格。

## 4. 最后写光影与质感
明确描述：

- 光源方向
- 光线颜色
- 明暗关系
- 整体色调
- 渲染风格与材质质感

# 输出要求
- **只输出最终提示词**
- **全中文**
- **不要解释、不要分析过程**
- **文字内容和字体描述必须置前**
- **避免空泛形容，尽量具体细致**
